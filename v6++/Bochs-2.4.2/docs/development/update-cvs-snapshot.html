5.2. Updating the CVS snapshot

The CVS snapshot can be updated with SF shell access using SSH. There is a script called update-snapshots.sh that can do all the required steps (checking out CVS, packing the source tree into one archive, updating the website link). 

ssh -t vruppert,bochs@shell.sourceforge.net create
